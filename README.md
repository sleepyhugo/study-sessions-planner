This is a full-stack web app I built for my Software Engineering Capstone. 
It helps users track study sessions, use a built-in Pomodoro timer, view dashboards, and schedule review reminders based on spaced repetition.

---

## Tech Stack
- **Backend:** Java 21, Spring Boot  
- **Frontend:** Thymeleaf, HTML, CSS, Javascript  
- **Database:** PostgreSQL  
- **Deployment:** Docker + Render
- **Other:** REST APIs, MVC architecture

---

## Features
- User signup & login (multi-user safe)  
- Log and track study sessions  
- Dashboard with charts  
- Pomodoro timer  
- Review reminders system  
- Dark mode  
- Modern unified navbar  
- Fully deployed online  

---

https://studysessions-app.onrender.com

---

## Running Locally
1. Clone repo
2. Set your PostgreSQL credentials in `application.properties`
3. Run the Spring Boot app
4. Open in browser `http://localhost:8080`
